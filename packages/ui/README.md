# `@workspace/ui`

shadcn/ui 기반 공유 UI 컴포넌트 패키지입니다.

## 사용법

```tsx
import { Button } from "@workspace/ui/components/button"
```

## Exports

| 경로                         | 내용               |
| ---------------------------- | ------------------ |
| `@workspace/ui/components/*` | UI 컴포넌트        |
| `@workspace/ui/hooks/*`      | 공유 훅            |
| `@workspace/ui/lib/*`        | 유틸리티 (`cn` 등) |
| `@workspace/ui/globals.css`  | 전역 스타일        |
